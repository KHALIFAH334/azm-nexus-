# AZM Nexus Landing Page 🏥

A premium, telemedicine-focused corporate landing page for **AZM Nexus Limited**. This project is built using **Next.js**, **Tailwind CSS**, and **Framer Motion** for a high-performance, interactive user experience.

## 🚀 Features

- **Telemedicine Specific**: Built from the ground up for healthcare technology.
- **Brand Identity**: Custom palette (Deep Teal, Active Lime, Soft Lavender).
- **Responsive Navigation**: Smart sticky navbar with a mobile hamburger menu.
- **Interactive Sections**:
  - **Solutions**: Virtual Care, Analytics, and Security showcases.
  - **Process**: 4-step clinical digitization methodology.
  - **Capabilities**: Technical infrastructure and compliance highlights.
  - **Contact**: Healthcare-ready contact form integrated with brand messaging.
- **Animations**: Smooth enter-animations and interactive hover states.

## 🛠️ Technology Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS (Custom Design System)
- **Animation**: Framer Motion
- **Icons**: Lucide-React
- **Language**: TypeScript

## 📦 Getting Started

### 1. Installation

Clone the repository and install dependencies:

```bash
cd azm-nexus
npm install
```

### 2. Local Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### 3. Production Build

```bash
npm run build
npm start
```

## 🌐 Deployment & Domain

### Vercel Deployment
The project is configured for seamless deployment to **Vercel**. Connect this repository to your Vercel dashboard for automatic CI/CD.

### Domain Transition
Once `azmnexus.com` is acquired:
1. Add the domain in the Vercel project settings.
2. Update the contact email in `src/app/page.tsx` from the temporary Gmail to your domain-based email.
